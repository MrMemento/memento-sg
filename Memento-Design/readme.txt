All WordPress hacks used by Memento Media.

== Description ==

A lot of hacks to make wordpress be more like a CMS than a blog engine, and also to hide WP related content on administration pages.

== Installation ==

1. Upload `Memento-Design` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
